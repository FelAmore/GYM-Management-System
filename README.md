# Gym Management System

A Java desktop application for managing gym operations, built with Java Swing 
and MySQL. Designed from the admin's perspective to simplify member 
registration and payment tracking.

## Features
- **Admin Login** — secure login page with show/hide password toggle
- **New Member Registration** — add members with personal details, gym 
  schedule, and monthly payment amount
- **Update & Delete Members** — search by member ID to edit or remove records
- **List of Members** — view all registered members in a table
- **Payment Tracking** — search by member ID to record and verify monthly 
  payments

## Tech Stack
- Java (Java Swing for GUI)
- MySQL (via JDBC)
- Object-Oriented Programming — Inheritance, Interface, Polymorphism

## How to Run
1. Clone this repository
2. Set up a MySQL database and configure the connection in `ConnectionProvider.java`
3. Open the project in NetBeans (or any Java IDE)
4. Run `Driver.java`
